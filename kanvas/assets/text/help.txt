Usage:
    kanvas
    kanvas create <name> [--type <template>]
    kanvas run
    kanvas help
    kanvas --theme <theme>

Commands:
    create      Create a new project in the current directory
    run         Run the project in the current directory
    help        Show this help message
    exit        Exit the interactive menu

Create options:
    --type      Project template: kanvas-sketch, java-app, java-lib, mixed-project

Themes:
    neon (default), mono, retro
