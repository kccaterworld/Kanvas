Usage:
    kanvas
    kanvas create <name> [--type <template>]
    kanvas list-types
    kanvas config [project] [overrides]
    kanvas run
    kanvas help
    kanvas --theme <theme>

Commands:
    create      Create a new project in the current directory
    list-types  Show available project templates
    config      Print the effective kanvas.json after CLI overrides
    run         Run the project in the current directory
    help        Show this help message
    exit        Exit the interactive menu

Create options:
    --type      Project template: kanvas-sketch, java-app, java-lib, mixed-project, empty

Config override options:
    --target, --encoding, --main-class, --jar-name, --package-version, --icon
    --src-dirs, --output-dir, --classpath, --dependencies, --native-targets

Themes:
    neon (default), mono, retro
